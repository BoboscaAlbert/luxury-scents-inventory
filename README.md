#Luxury Scents - Management System (Java)

Un sistem de gestiune a inventarului pentru un magazin de parfumuri, creat în Java. Proiectul demonstrează concepte fundamentale de Programare Orientată pe Obiect (OOP) și persistența datelor.

 Funcționalități
* **Autentificare:** Sistem de login pentru administratori.
* **Gestiune Inventar:** Adăugare, vizualizare și vânzare de produse.
* **Categorisire:** Suport pentru parfumuri de Nișă, Designer și Arabești.
* **Persistență:** Salvarea automată a datelor într-un fișier `inventar.txt` (format CSV).
* **Validări:** Prevenirea vânzării produselor care nu sunt în stoc (Out of Stock).

 Tehnologii folosite
* **Limbaj:** Java 17+
* **Concepte:** OOP (Encapsulation, Polymorphism), File I/O, Exception Handling, Collections (ArrayList), Lambda Expressions.

 Cum se rulează
1. Clonează repository-ul.
2. Deschide proiectul în IntelliJ IDEA sau orice IDE Java.
3. Rulează clasa `Main`.
4. Date de acces: Utilizator: `admin` | Parolă: `scent123`.
